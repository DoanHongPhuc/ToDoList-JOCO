# Overview
A Web App is a product created for learning purposes. It allows users to take notes and manage their tasks on a daily basis.
- Today
![Today Screen overview](./assets/Today.png)
    - Create new task (Task_name, Description,Due Time, Priority, Label)
    - Edit task
    - Check Task
    - Delete Task
## Installation
```bash
$ yarn install
```
## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
